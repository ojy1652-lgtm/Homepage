# Research Lab Homepage

Astro and Decap CMS based research laboratory homepage.

https://em3-homepage.netlify.app/

## Pages

- Home
- Research
- Publications
- People
- Facilities
- News

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## CMS

The Decap CMS admin page is available at `/admin/`.

Content is stored as Markdown under `src/content`, and uploaded images are stored under `public/images/uploads`.
